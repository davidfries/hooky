Get started (UI & CLI)

UI Quickstart

# Run with Docker
$ docker run -p 3000:3000 ghcr.io//hooky:latest

# Or run from source (Deno)
$ deno task server

# Open the UI
http://localhost:3000

# Create an endpoint
$ ./hooky create

# Send a test payload (replace )
$ curl -X POST -H 'content-type: application/json' \
  -d '{"demo":123}' http://localhost:3000/hook/

CLI (Deno)

# Download release binaries
# https://github.com/davidfries/hooky/releases

# Create (10 min TTL example)
$ ./hooky create 600

# List endpoints
$ ./hooky list

# List events
$ ./hooky events 

# Ping endpoint
$ ./hooky ping  '{"demo":123}'
